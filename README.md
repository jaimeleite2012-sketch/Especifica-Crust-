# Dashboard Crust Itália · Qualidade

Dashboard mobile-first gerado a partir da planilha matriz:

**BRM-MDL-SQ-SGI-033 · Rev.23 · Especificação Crust Itália**

## O que foi incluído

- Tela principal com busca rápida por artigo, cliente, TR, cor e receita.
- Ficha técnica por artigo com TR, cor, código de cor, cliente, yield, espessura, receita, fluxo e características finais.
- Explosão visual do fluxo de secagem/processo ao clicar no artigo.
- Aba de parâmetros de máquinas baseada na aba `ESPECIFICAÇÕES Arianna`.
- Aba de fornecedores com status aprovado, reprovado e em teste.
- Histórico de revisões da folha de aprovação.
- Botão para copiar relatório técnico do artigo selecionado.
- PWA simples para uso em celular, com `manifest.webmanifest` e `sw.js`.

- Artigos: 19

- Fornecedores aprovados na matriz: 34

## Publicação recomendada no GitHub Pages

1. Crie um repositório separado, preferencialmente privado se a informação for sensível.
2. Nome sugerido: `dashboard-crust-qualidade`.
3. Envie todos os arquivos deste pacote para a raiz do repositório.
4. No GitHub, acesse `Settings > Pages`.
5. Em `Build and deployment`, selecione `Deploy from a branch`.
6. Branch: `main`; pasta: `/root`.
7. Acesse a URL gerada pelo GitHub Pages.

## Atenção de confidencialidade

Este pacote contém dados técnicos extraídos da matriz. Antes de publicar em um repositório público, valide a política interna de confidencialidade da empresa.
